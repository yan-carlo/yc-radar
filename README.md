# yc-radar
Radar militar-style para aeropuertos en FiveM. Incluye detección configurable por radio y altitud mínima, interfaz HTML/CSS/JS grande y responsive con direcciones cardinales, zoom, opción para ocultar aeronaves específicas, sistema de transponders (encendido/apagado), blips unificados sin distinguir NPC/jugador, localización (es/en)
